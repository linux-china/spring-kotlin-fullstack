Kotlin FullStack Demo

Spring WebFlux Kotlin fullstack example with Kotlin/JS for frontend instead of JavaScript or TypeScript.

*  Kotlin/JS for frontend instead of JavaScript or TypeScript

### Back end

* Spring WebFlux
